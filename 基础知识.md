## 重要语句和导出数据类型

### 条件判断语句

* if语句  
    ```cpp
    if (<表达式>) {
        <语句1>
    }else{
        <语句2>
    }
    ```
* switch语句
    ```cpp
    switch (<表达式>) {
        case <情况常量1>:
            <语句1>
            break;
        case <情况常量2>:
            <语句2>
            break;
        ...
        default:
            <语句n>
    }
    ```

### 循环语句

* for循环语句
    ```cpp
    for (int i = 0; i < n; i++) {
        <语句>
    }
    ```

* while循环语句
    ```cpp
    while (<表达式>){
        <语句>
    }
    ```

* do-while循环语句
    ```cpp
    do {
        <语句>
    }while(<表达式>)
    ```

## 函数

### 函数声明与调用
* 声明
    ```cpp
    void print (int a) {
        cout << "Hello World!";
        cout << a + 5;
    }
    ```

* 调用
    ```cpp
    print(4);
    ```

* 参数引用调用  
    * 需要改变变量值 

    * 对占内存多的参数，节省内存。此情况为不修改，可在形参前加const。

* 函数重载

* 运算符重载（operator<运算符>）
    ```cpp

        if((a==TRUE)&&(b==TRUE)) {
            return true;
        }
        return false;
    }
    ```
